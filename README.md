## 📖 About Project
**ClassicOS 7.5** is a high-performance system shell developed entirely in Python. Unlike basic wrappers, this project features a **fully custom-drawn interface** directly from the code. It replicates the classic skeuomorphic era with modern rendering techniques, optimized specifically for **Windows 10 and 11**.

Every button, window frame, and glass texture is calculated and rendered programmatically, ensuring a unique and lightweight experience.

## ✨ Key Technical Features

* 🐍 **Procedural UI:** The interface isn't just images—it's drawn via code (Qt Painter/QSS), allowing for perfect scaling and sharp visuals.
* 🪟 **Windows 10/11 Exclusive:** Leverages Windows-specific APIs for deep system integration.
* 💎 **Skeuomorphic Engine:** Real-time rendering of shadows, glass blur (Aero style), and 3D-effect buttons.
* ⚡ **High Performance:** Optimized Python logic that keeps CPU usage low while maintaining smooth animations.
* 📂 **Integrated File System:** A custom-built explorer to manage your files within the ClassicOS environment.
